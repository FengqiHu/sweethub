# SweetHub

#### 介绍
本项目来源于LikeGirl开源项目，并在其基础上进行了功能上的修改，并更名为SweetHub，支持了本地上传图片以及压缩，有效提升了加载速度。

感谢原作者的贡献，前人栽树后人乘凉。为尊重原作者，下面是原作者的声明，感谢！
